About This Bot

A professional Telegram inline bot built with TypeScript that allows you to search GitHub repositories without leaving your chat.

Fast Search

Instant repository search with intelligent caching

Smart Results

Find users and specific repositories with ease

Optimized

Built for high-volume usage with rate limit management

Start Using Bot

How to Use

Search User Repositories

@githubcom_bot microsoft

Shows all repositories from Microsoft

Search Specific Repository

@githubcom_bot microsoft/vscode

Shows the specific Visual Studio Code repository
